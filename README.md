# neuverdrahtet Website – Deployment & SEO Guide

## 🚀 Schnellstart: GitHub Pages

1. GitHub Account erstellen: github.com
2. Neues Repository: "neuverdrahtet-website" (public)
3. Alle Dateien hochladen
4. Settings → Pages → Branch: main → / (root) → Save
5. Custom Domain eintragen: neuverdrahtet.com
6. "Enforce HTTPS" aktivieren
